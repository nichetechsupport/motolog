# MotoLog サポート

お問い合わせの前に、以下のよくある質問をご確認ください。

## アプリについて

MotoLog は、バイクオーナー向けの整備記録・給油記録アプリです。オイル交換やチェーン清掃などの整備時期を通知でお知らせし、給油記録から燃費を自動計算します。データはすべて iPhone 内に保存され、アカウント登録は不要です。

## よくある質問

### データの保存について

**Q. データはどこに保存されますか？**
すべて iPhone 本体のローカルストレージに保存されます。サーバーには何も送信されません。

**Q. 機種変更のときにデータを引き継げますか？**
iCloud Drive を有効にしている場合、設定画面の「iCloud にバックアップ」を実行してから、新しい端末で同じ Apple ID にサインインし、本アプリから「バックアップから復元」を実行するだけで移行できます。iCloud Drive を使っていない場合は、設定画面の「ファイルに書き出し（JSON）」で作成したバックアップファイルを AirDrop やメールで新しい端末に送り、復元することも可能です。

**Q. アプリをアンインストールしたらデータはどうなりますか？**
iOS の仕様により、アンインストールするとすべてのデータが消えます。大切なデータは必ず事前にバックアップしてください。

### Pro 版について

**Q. Pro 版と無料版の違いは？**

| 機能 | 無料 | Pro |
|---|---|---|
| バイクの登録 | 1台まで | 何台でも |
| 整備項目のカスタマイズ | × | ✓ |
| 項目ごとの通知設定 | × | ✓ |
| 費用の記録と集計 | × | ✓ |
| 分析ダッシュボード (月別コスト、走行距離、燃費チャート) | × | ✓ |
| PDF / CSV 書き出し | × | ✓ |
| 整備記録への写真添付 | 2枚まで | 5枚まで |

Pro 版は ¥900 の一度きりの買い切り購入です。

**Q. Pro 版を購入した Apple ID を変えたい / 機種変更したい場合は？**
Pro 購入画面の「購入を復元」ボタンから、同じ Apple ID での購入履歴を復元できます。

**Q. Pro 版を購入しましたが、返金してもらえますか？**
App Store での購入の返金は Apple が管理しており、当方では対応できません。Apple の [購入に関する問題の報告ページ](https://reportaproblem.apple.com/) から返金をリクエストしてください。返金の可否は Apple の判断となります。

**Q.「購入を復元」を押しても Pro が有効になりません。**
以下を確認してください:

- iOS の設定アプリで、Pro 版を購入したときと同じ Apple ID でサインインしているか
- インターネット接続が有効か
- それでも解決しない場合は Apple のサポートにお問い合わせください

### 通知・リマインダーについて

**Q. 通知が来る時刻を変えたい。**
設定タブ内の「通知時刻」から変更できます。初期設定は朝 9 時です。

**Q. 整備通知が届きません。**
以下を確認してください:

- iOS の設定アプリ → MotoLog → 通知 が「許可」になっているか
- バイクの整備項目ごとに「通知」が有効になっているか (Pro 版のみ切替可)
- その項目に整備記録が 1件以上あるか (通知は前回の記録を基準に次回予定を計算します)
- 次回予定日の「〇日前」に通知する設定が有効か

なお、次回予定日をすでに過ぎている場合は通知が送信されません。新しい整備記録を追加することで、次回予定日が再計算されて通知が再スケジュールされます。

### 機能について

**Q. 燃費が表示されません。**
燃費は給油記録が 2件以上ある場合に計算されます。また、走行距離 (オドメーター) が入力されている給油記録が必要です。

**Q. オドメーター (走行距離) を修正したい。**
バイク詳細画面のオドメーター表示をタップすると、手動編集できます。

**Q. 間違えて記録を削除してしまいました。**
現バージョンでは、削除した整備記録を復元する機能はありません。重要な記録は削除前に内容をよくご確認ください。

**Q. 整備項目を独自に追加したい。**
Pro 版で、バイクごとの整備項目設定画面から独自の項目を追加できます。

### バックアップ / 復元

**Q. バックアップはどこに保存されますか？**
バックアップの保存先は、ご利用の端末の iCloud Drive 利用状況に応じて自動的に決まります:

- **iCloud Drive が有効な場合**: お使いの Apple ID に紐づく iCloud の本アプリ専用領域に保存されます。同じ Apple ID でサインインした他の端末から復元できます。
- **iCloud Drive が無効な場合**: 設定画面のボタンは「ファイルに書き出し（JSON）」に切り替わり、iOS の共有シートから保存先 (ファイルアプリ、AirDrop、メール等) を自由に選べます。

**Q. iCloud バックアップを使いたいのですが、ボタンが表示されません。**
iOS の設定アプリで iCloud Drive を有効にしてください。「設定 → [ユーザー名] → iCloud → iCloud Drive」をオンにした後、アプリを開き直すと「iCloud にバックアップ」ボタンが表示されます。

**Q. iCloud バックアップは自動で行われますか？**
いいえ、自動ではありません。設定画面の「iCloud にバックアップ」ボタンを押したときにのみ実行されます。最新の状態を保ちたい場合は、定期的に手動で実行してください。

**Q. 古いバックアップから復元できますか？**
はい、バックアップファイルには互換性のためのバージョン情報が含まれており、アプリが対応可能なバージョンであれば復元できます。

**Q. 復元したら元のデータはどうなりますか？**
復元操作はデータを置き換える形で行われます。操作前に現在のデータのバックアップを取ることをおすすめします。
