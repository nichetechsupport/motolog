# プライバシーポリシー

**最終更新日:** 2026年4月16日

NicheTech Support (以下「当方」) は、iOS アプリ **MotoLog** (以下「本アプリ」) におけるユーザーのプライバシーを尊重し、以下のとおりプライバシーポリシーを定めます。

## 1. 基本方針

本アプリは **ローカルファースト** 設計です。ユーザーが本アプリに入力・保存するすべてのデータは、原則としてユーザーの iPhone 内にのみ保存され、当方のサーバーや第三者には送信されません。

## 2. 収集する情報

本アプリは、アカウント登録や個人情報の送信を一切必要としません。当方はユーザーを識別できるいかなる情報も収集しません。

ユーザーが本アプリに入力する情報 (バイクの情報、整備記録、給油記録、写真、メモ等) は、すべてユーザーの iPhone のローカルストレージにのみ保存されます。

## 3. データの保存場所

| データ | 保存場所 |
|---|---|
| バイク情報、整備記録、給油記録、写真、アプリ設定 | ユーザーの iPhone 内 (SwiftData / ローカルストレージ) |
| 通知スケジュール | ユーザーの iPhone 内 (iOS の通知システム) |
| Pro 版購入状態 | Apple の StoreKit および本アプリ内部記録 |
| iCloud バックアップ (ユーザーが明示的に実行した場合) | ユーザーの Apple ID に紐づく iCloud Drive (本アプリ専用領域) |
| ファイル書き出しによるバックアップ | ユーザーが共有シートで選択した保存先 |

## 4. 第三者への情報提供

当方は、ユーザーのデータを第三者に販売・貸与・提供することは一切ありません。

本アプリには、解析ツール (Google Analytics、Firebase Analytics 等)、広告 SDK、トラッキング目的のサードパーティ SDK は一切組み込まれていません。

## 5. トラッキングについて

本アプリは、App Tracking Transparency (ATT) の対象となるトラッキングを一切行いません。ユーザーのアクティビティを他社アプリや Web サイトを横断して追跡することはありません。

## 6. 通知について

本アプリは、ユーザーが設定した整備時期をお知らせするためにローカル通知 (iOS の UserNotifications フレームワーク) を使用します。通知スケジュールはユーザーの端末内のみで管理され、当方のサーバーを経由することはありません。

通知の許可はいつでも iOS の設定アプリから変更できます。

## 7. 写真の取り扱い

バイクの写真や整備記録の写真を添付する際、iOS の写真ライブラリまたはカメラへのアクセスが必要です。これらの写真はユーザーの iPhone 内にのみ保存され、外部に送信されることはありません。

## 8. 購入情報について

Pro 版の購入には Apple の StoreKit を利用します。決済情報 (クレジットカード番号等) は Apple が管理し、本アプリおよび当方はこれらの情報を取得しません。

購入の事実 (Pro 版が有効か否か) のみが、Apple のシステムを通じて本アプリに通知されます。

## 9. バックアップ機能について

本アプリは、ユーザーが設定画面から明示的に実行した場合にのみ、データ全体を JSON 形式でバックアップします。バックアップの保存方法は以下の 2 種類です。

### 9-1. iCloud へのバックアップ

ユーザーが「iCloud にバックアップ」を実行すると、バックアップファイルは本アプリ専用の iCloud Drive 領域 (App Container) に書き込まれます。このデータは Apple の iCloud サービスを経由してユーザー自身の Apple ID に紐づく iCloud に保存され、同じ Apple ID でサインインした他の端末 (機種変更後の iPhone 等) から復元することができます。

- iCloud に保存されたバックアップデータは、Apple により送信時および保存時に暗号化されます。
- 当方 (NicheTech Support) は、ユーザーの iCloud に保存されたデータにアクセスする権限を一切持ちません。
- iCloud バックアップを利用するには、iOS の設定で iCloud が有効になっている必要があります。
- 詳細は [Apple の iCloud プライバシーについて](https://www.apple.com/jp/legal/privacy/) をご参照ください。

### 9-2. ファイル書き出しによるバックアップ

ユーザーが「ファイルに書き出し」を選択した場合、バックアップは一時ファイルとして作成され、iOS の共有シートが表示されます。ユーザーは保存先 (ファイルアプリ、AirDrop、メール等) を任意に選択できます。

- 書き出したバックアップファイルはユーザーが選択した保存先にのみ送信されます。
- 当方がこのファイルを受け取ることはありません。

## 10. お子様のプライバシー

本アプリは 13 歳未満のお子様を対象としていません。当方は意図的にお子様の個人情報を収集することはありません。

## 11. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更や本アプリの機能追加に応じて改定することがあります。重要な変更があった場合は、本ページ上で告知します。
